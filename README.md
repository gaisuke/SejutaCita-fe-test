# Online Test - Frontend Engineer

This repo is for doing Online Test - Frontend Engineer Test from SejutaCita (PT SejutaCita Anak Muda Indonesia)<br />

## Installation

Clone the repo.

```bash
git clone https://github.com/gaisuke/SejutaCita-fe-test.git
```

Install the package

```bash
npm install
```

Run project

```bash
npm start
```

## Tech Stack

- ReactJS
- Redux-toolkit
- React Router
- TailwindCSS
